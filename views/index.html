Identificacion de postulante usando base de datos MongoDB en nube usando Openshift

En este proyecto de Sistemas Distribuidos simularemos el registro de un postulante en una base de datos NoSQL

para ser exactos utilizaremos MongoDB

Estos datos podrán ser consultados, desde la pagina de consultas

La Coleccion de datos que guardaremos lo llamaremos Postulante

y el Documento o esquema a utilizar es el siguiente.

Atributo Tipo Descripción Importancia
DNI Numerico Número de DNI del postulante Alta - no puede ser nulo
Nombre String Guardara el nombre del postulante Alta - no puede ser nulo
Apellido String Guardara el nombre del postulante Alta - no puede ser nulo
Correo Email Guardara el correo del postulante Puede ser nulo para registrar
Telefono Numerico Guardara el número telefonico del postulante Puede ser nulo para registrar
Todos los derechos reservador
